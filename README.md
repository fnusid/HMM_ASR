# README
This is a basic HMM based ASR which uses MFCC and delta values as features to identify 6 phrases 
 
1. 'Odessa' : Keyword phrase, 

2. 'Play music'

3. 'Start music'

4. 'Turn off the lights'

5. 'Turn on the lights'

6. 'What time is it'

## Dependencies
`librosa, numpy`
